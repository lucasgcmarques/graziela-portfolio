# Graziela Paes Portfolio

Artistic website and portfolio for **VJ Grazzi** (Graziela Paes).

## About

A digital showcase for VJ and visual art works by Graziela Paes.

## Tech Stack

- Next.js
- React Three Fiber
- TailwindCSS
- TypeScript

## Development
